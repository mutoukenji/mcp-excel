# Excel MCP 工具 · 需求文档

## 1. 这个产品是什么

一个让 AI 助手（比如 Kimi、Claude 等）能直接读取和编辑 Excel 文件的"小插件"——装上它之后，您只需要对 AI 说一句大白话，AI 就能帮您打开、查看、修改电脑上的 Excel 表格。

> 背景说明：MCP 是一种通用的"AI 插件标准"。这个产品本身没有自己的软件界面，它是装进 AI 助手里使用的。类似的参考产品是 office-powerpoint-mcp-server（一个让 AI 操作 PPT 的工具），我们做的是 Excel 版。

## 2. 谁会用它

- 任何使用 AI 助手、并且需要处理 Excel 文件的人。
- 典型用户：办公人士、财务/行政人员、数据整理者，以及所有"想让 AI 帮忙干表格活"的普通用户。
- 面向所有人公开，不设门槛、不需要注册账号、不需要管理后台。

## 3. 功能清单

### 3.1 没它不行的核心功能

| 功能 | 通俗解释 |
|------|---------|
| 读取指定文件 | 用户告诉 AI 文件放在哪，AI 就能把表格里的内容读出来、讲给您听 |
| 创建 Excel 文件 | 从无到有新建一个 Excel 文件 |
| 编辑任意 Sheet | 对文件里的任何一张工作表（Sheet）进行增、删、改：加一张表、删一张表、改某个单元格的内容 |
| 支持所有常见格式 | 老版的 `.xls`、新版的 `.xlsx`、以及 `.csv` 等常见表格格式都能处理 |
| 插入图片 | 能把图片（比如产品图、Logo、截图）放进表格里 |
| npx 一键安装 | 用户只需要复制一条命令，就能完成安装，不用懂技术 |

### 3.2 有了更好的重要功能

| 功能 | 通俗解释 |
|------|---------|
| 写入公式 | 能在单元格里放计算公式（比如求和、平均值），让表格自己算数 |
| 美化格式 | 设置字体、颜色、边框、合并单元格、调整列宽行高，让表格好看 |
| 数据处理 | 按条件筛选数据、排序、去重等常见整理操作 |
| 生成图表 | 根据表格数据生成柱状图、折线图等，直观展示 |
| 批量操作 | 一次写入一大片数据，而不是一个格子一个格子地填 |

### 3.3 锦上添花的功能

| 功能 | 通俗解释 |
|------|---------|
| 多文件合并 | 把好几个月的报表自动合并成一个文件 |
| 数据校验 | 限制某个格子只能填数字或日期，防止填错 |
| 模板能力 | 提供常用模板（如月度报表、考勤表），快速生成 |
| 导出其他格式 | 把表格转成 PDF 等格式，方便分享 |

> 说明：3.2 和 3.3 是在您核心需求基础上补充的建议项，开发时可以按优先级取舍。

## 4. 额外要求

- **全平台可用**：Windows、Mac、Linux 三个系统都能安装运行。
- **安装要极简**：通过 `npx` 一条命令安装，用户不用配置复杂环境。
- **数据不出本机**：所有文件都只在用户自己的电脑上处理，不上传到任何云端服务器，保证隐私安全。
- **公开免费使用**：面向所有人，无需注册、无需登录、无管理后台。
- **AI 能理解**：工具的每个功能都要描述得清清楚楚，让 AI 一看就知道什么时候该用哪个功能。

## 5. 典型使用场景

- **场景一：看表**
  小王是一名销售，电脑上有一个《2026年销售记录.xlsx》。他对 AI 说："帮我看看这个文件里的'一月'那张表，告诉我总销售额是多少。"AI 通过本工具读取文件，算出结果告诉他。

- **场景二：建表**
  小李要做一份员工考勤表。她对 AI 说："帮我新建一个 Excel，里面分 12 张 Sheet，每张是一个月的考勤表，第一行写上姓名和 1 到 31 号。"AI 通过本工具创建文件并填好内容。

- **场景三：改表**
  财务小张发现报表里有个数字错了。她对 AI 说："把'费用表'里 B5 单元格的 3000 改成 3500，再把最后加一行合计。"AI 直接修改文件保存。

- **场景四：插图**
  电商运营小赵在做商品清单。他对 AI 说："把这几张产品图插到表格对应商品的旁边。"AI 把图片插入到指定位置。

## 6. 长什么样子

这个产品和普通软件不一样——**它自己没有窗口和页面**，它是"住"在 AI 助手里面干活的。用户实际会接触到的是这三样东西：

1. **安装页面**：一个说明页（npm 包页面），上面有一段安装命令和配置方法，用户照着复制粘贴就能装好。
2. **AI 对话窗口（用户的主战场）**：用户在自己常用的 AI 助手里用大白话提需求，比如"帮我读一下桌面上的表格"。AI 在后台调用本工具完成任务，然后把结果用对话的形式告诉用户。
3. **成果文件**：任务完成后，用户在电脑上打开自己的 Excel 文件，就能看到读取的结果、修改后的内容、插入的图片。

一句话总结：**用户感觉不到这个工具的存在，只觉得"AI 突然会操作 Excel 了"**——这就是它最好的样子。
